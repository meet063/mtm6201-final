#  Plant Palette – Web Portfolio
##  Project Overview
This project is a responsive website created using HTML, CSS, and Bootstrap.  
The goal of this project was to design and develop a modern plant-based restaurant website based on my Figma design.

The website includes multiple pages:
- Home
- About
- Menu
- Contact

---

##  Design Process
I started by creating wireframes and mockups in Figma.  
After finalizing the design, I used Bootstrap grid system and custom CSS to build a responsive layout.

I focused on:
- Clean and modern design
- Mobile-first approach
- Consistent color scheme using CSS variables

---

##  Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Google Fonts (Playfair Display & Roboto)
- Bootstrap Icons

---
## Accessibility
I applied accessibility best practices such as:
- Skip link for keyboard navigation
- Alt text for images
- Proper heading structure
- Form labels for inputs
